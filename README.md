# Simple-Notes-App

Ini adalah aplikasi catatan sederhana berbasis web dengan [ReactJs](https://react.dev/).
## Demo 
 [Notes](https://note-app-vite-three.vercel.app/)
## Features

- Create Note
- Edit Note
- Delete Note
- Search Notes
- Dark-Light Mode



## Installation

### Clone Project

```bash
 git clone https://github.com/Gibrand22/note-app-vite.git
```
```bash
 cd note-app-vite
```
### Install Dependensi
```bash
 npm install
```
### Run development
```bash
 npm run dev
```
